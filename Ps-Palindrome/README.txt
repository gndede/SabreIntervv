ONLINE 47 to Line 51 we manually load the program with words which are either Palindrome, not Palindrome or blank space.
When the program is run, it is able to determine is the word is 

a) Blank Space
b) Is a Palindrome
c) Not a Palindrome

eg.

  - when no palindrome exists => 'No Palindrome exists'
  - When there is only one palindrome => 'No Second Palindrome exists'
  - When there is a second palindrome => 'Found Palindrome: [THE PALINDROME WORD]'
  
  Directions on how to run the program:
  
  i) Download this file in a directory of your choice
  ii) Make sure it is saved as .js file
  iii) Navigate to the directory from the command prompt cd <directory name>
  iv) Execute the following command c:\DirectoryName\nodepalindrome.js
  
  On the screen it will display options on whether a word is a Palindrome, Not a Palindrome or Blank.
